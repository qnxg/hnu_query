# 项目结构

项目的根目录下包含：

* `docs`：本项目的文档
* `examples`：一些示例代码
* `src`：项目的代码
* `.env`：用于测试的环境变量，见 `docs/test.md`

其中 `src` 中，每一个文件夹都表示学校的一个系统。名称一般为其请求的接口的子域名名称，比如需要请求 `hdjw.hnu.edu.cn` 的接口都放在 `src/hdjw` 下。

* `ca`：可信电子凭证平台
* `cas`：统一身份认证
* `gym`：体测系统
* `hdjw`：教务系统
* `lab`：大物实验平台
* `netflow`：校园网流量系统
* `pt`：个人门户
* `wxpay`：目前只包含了宿舍电量查询的接口
* `xgxt`：学工系统

学校系统对应的文件夹下，一般会包含 `login.rs` 或同名的模块，这里面存放的是登录该系统的代码。然后还会有其他若干模块，这些模块往往按接口的主题进行划分。
